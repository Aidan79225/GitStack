## Known additions to fold in (from theme PR #55)

- **Light theme is primary-tinted.** Surfaces are no longer pure white.
  Confirmed: `light.json` sets `"surface": "#eff5fd"`.
- **Custom panel inspection.** Light/Dark/System users can open the
  Theme dialog → Custom radio to expand sections and read each token's
  hex code without entering edit mode. Confirmed: `_open_picker()` in
  `theme_dialog.py:291` is a no-op unless `_selected_mode() == "custom"`.
  The panel is always rendered (not hidden), so it serves as a read-only
  inspector in any non-Custom mode. Toggling radio buttons refreshes swatches live.
- **Typography scale is global.** The slider's effect persists across
  restarts in any mode (System/Light/Dark/Custom), driven by the
  `typography_scale` setting. Confirmed: `settings.py:11` sets default,
  `manager.py:73` reads it at apply time, `theme_dialog.py:338` saves it.
